# Application de gestion des sinistres – CNMA

## Description
Application web développée en PHP/MySQL permettant la gestion complète des sinistres automobiles.

## Fonctionnalités
- Gestion assurés
- Gestion tiers
- Gestion contrats
- Création dossier sinistre
- Expertise
- Gestion des réserves
- Gestion des règlements
- Historique des actions
- Validation CNMA
- Clôture dossier

## Technologies
- PHP
- MySQL
- HTML/CSS
- JavaScript
- XAMPP

## Installation
1. Importer le fichier database/bdd.sql dans phpMyAdmin
2. Modifier cnma/includes/config.php
3. Lancer Apache et MySQL
4. Ouvrir http://localhost/PfeCnma
